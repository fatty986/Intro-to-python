# Intro to Python

This repository contains **Mission 1 – Python Basics**.

## Mission 1 – Basics
This mission introduces:
- variables
- arithmetic expressions
- conditional statements
- simple loops

### Structure
- `pre/` → warm-up exercises
- `core/` → main exercises
- `post/` → harder exercises (later)

Each exercise folder contains:
- a `README.md` with the task description
- a `solution.py` file to complete
- automatic tests
